選手歷程(標題我不知道該怎麼打)
===
大概訴說選手這段時間發生的事。有些記憶是零碎的
，大概會想到什麼打什麼。


## 動機

> 這是我在高三畢業的暑假打得一篇長文(大概拉)，
> 因此有些事都已經忘記了。


## 高職前的時間 ( 9月前 ) 

在上國中之前，我就知道對資訊方面的技術非常有興趣。
因為爺爺是為技士，可能是因為耳濡目染的關係。
會覺得那些東西非常的酷，如果學會之後就可以拿出去炫耀XD。

在國中我算是一個非常安靜的人吧，不太會關係班級事務的人。


## 為什麼會參加選手 ( 9月 )

當我進入羅東高工的時候，就開始宣傳當選手怎樣怎樣，可以碰酷酷的設備，
也能讓老師優先認識你...反正那時候就有什麼好處都說這樣，但那些都沒有打動我
那時候只抱持著很簡單的想法，去玩玩看好了，反正大不了又會跟國中一樣
一定玩不了太久，況且也不知道當選手具體是要幹嘛。

那時候有兩個最主要的職類：
* 資訊網路技術
> 完全不了解的職類，只知道好像是設定網路那些的職類，
> 感覺像很簡單又好像很難的樣子

* 機器人
> 大概清楚的職類，好像是寫機器人的職類
> 需要會寫程式，組機器人的樣子，老師一直說搬機器人的人會很累
因為他很重

### 岔開話題.1

那時候記得是上機器人的課程，就在大間的電腦教室，每個人都在寫機器人的程式
，而那時候老師突然打斷我們，說：

> 大家我把電腦切過來，來看選手頒獎 ( 播著高雄頒獎的畫面，那時候好像是陳菊頒獎的樣子 )
，直到學長掛上金牌之後，老師開始大肆宣傳，他金牌之後就能在學校每天掛著金牌的布條
之類之類的。

但那時候沒有太多的感覺，那時候只感覺就金牌阿，學長很厲害並沒有什麼特別的感覺。
但他卻深深影響未來的資訊科選手

### 岔開話題.2

我住在宜蘭，而學校是在羅東，所以我們學校會有專車的樣子，在當選手之後
就不能搭專車，因此我很羨慕那些可以4點回家的人。可以早一點看到多一點太陽
，也能早一點回去玩遊戲

等等這篇文會不會變超長，因為才講1/10而以哈哈。


## 培訓的時候 ( 10月~12月 ) 
---
首先剛開始的時候，有分成兩個最主要的兩個培訓職類，
資訊網路技術、機器人，而又有兩個教室，一個是學長待的選手室，
一個是我們培訓的教室，那時候都對學長非常尊敬，因為不管他們有沒有得名
，都會覺得他們很厲害，因為不只懂那些艱深的知識，還有能進去那未知的全國賽


* **資網培訓**

雖然培訓的時候有兩個職類，但我們都在同一個教室裡面，所以當機器人有練習的時候
我們都會偷偷地看他們在幹嘛，而且剛開始的一個禮拜還跟主任說，能不能換職類，從資網換去機器人那邊
因為機器人真的比較好玩，可以看到實體的機器動作，比資網好太多了，

資網就是簡單來說，把網路上的設備連通並做出一些特別的功能，跟機器人差太多了
資網是虛擬的東西，而機器人是實體的，所以比較好玩

而主任說，你只要在一個禮拜內想換都可以，好玩的是 那時候是我提的主意，
但我自己沒換，其他人到是換了很多。

開始練習資網許久後發現隔壁的仁傑突然變強很多，
把那時候很多不懂得技術都搞得清楚，因此那時候還會去問他
但給我的感受是，我真的很不適合資網，要不要退出選手，
那時候已經有很多人都退出了選手，有些人是用距離太遠回家太晚，
有寫人是成績下降的原因

但我是單純坐不住這麼久的椅子，每天要待在椅子上還要思考資網技術類的東西
太不適合我了，想起來動一動的職類，但那時後真的沒有適合的。


* 突如其來的布建並參加了

> 在最後選拔的時候突然有布建這個職類，但只比過一屆所以大家都不知道
到底在幹嘛的，但我想說 與其退出選手還不如試試看其他職類，於是就參加了布建

## 準備比賽前 ( 12月~3月 )
---

那時候的布建是待在另外個小教室，跟資網和機器人不同的選手室，
因此我跟峻睿只有在修休息的時候才會去選手室，
其他時間都在練習時間划手機，或者做小小的網路頭練習，
那時候只能照著去年的國手賽題目練習。

話說我們那時候只有一位學長會回來，而且只有星期六的時候，好像是因為
他已經大學了，具體也不知道為什麼會回來比

而我們在練習的時候也陸陸續續的進了很多新的設備，比如說：熔接機、室外光纖
...真的很多東西，連我自己也嚇一跳，之後知道價格後更差點嚇死XDD
，有一次還被指導老師(實習主任)叫去修充電線

## 電訊布建的集訓 ( 4月 )
---

這是我第一次體會選手的集訓，這個感覺非常的不一樣，看到學長為了自己的比賽而
努力，集訓的每一個小時都沒有浪費的。雖然我們跟他們不同教室，但還是能受到一些影響。

* 分區賽說明會<br>
這是我第一次參加說明會，感覺不用特別準備什麼，指導老師他們好像也會去，
只是不是跟我們一起，因此我跟峻睿兩個人就獨自去台中，那時候是上課時間，還跟同學用IG聊天
，我跟峻睿坐在麥當勞那邊聊天，那時候時間估算大提早，每一個環節都辜半小時的誤差，
路程是高鐵->公車，但中途的公車停駛就在麥當勞先吃個午餐，還記得我們是從麥當勞走去職訓局那邊，
中途都是大卡車急速，到那邊後1個小時才看到我們的指導老師慢慢地現身，最後要走的時候賴老師也跟蔡老師聊天，
最後我們是跟賴老師搭計程車會去高鐵站，為甚麼我們不早一點搭計程車，要慢慢地搭公車還要用走的，有夠累ww。

* **布建訓練的過程**<br>
那時候要比什麼樣的東西都還不知道，因此只能照著去念的國手賽，為什麼只有
參考國手賽，不參考其他的題目呢。聽說那時候以為是50屆才開始比布建的比賽，但
勞動部臨時跟裁判長說49就要開始比，因此那屆只有國手賽。

那時候的集訓剛開始還很興奮，還記的那時候連資網也搬過來跟我們一起集訓，
因為整間選手室都要給機器人用。而看到同學們對面教室上課，有時候我也會跟他們打招呼，
感覺還滿好玩的。

具體真的要說練習的過程罵，現在回想起來非常的枯燥，每天都是到學校後，早上8點開始練習
沒有在學校吃早餐的，因此壓力非常的大，雖然那時候的我都晚上8點走，甚至還有學長不滿我那麼早走哈哈哈
最後還不是比他們好(沒)。

記得有一次非常的晚走，大概10點多得時候吧，那是因為我有一些機架的布線沒有練到自己想要的美觀
，因此就留下來練習，現在回想起來真的是我選手生涯裡面最晚走的時候了。我還依稀季的，也有一次是大家一起離工廠，
看著鐵門拉下來的一刻。最後出電梯的時候還看到對面工廠的電梯也是打開的，完全沒有關上，真的是嚇死我了。

* 楠億的朋友<br>
這也算個小插曲，這讓我認識到了，選手真的是很忙的一個族群，
還有一個很閒的人配上很忙的人真的不適合。<br>
就有一次晚上八，好像是我幫楠億拍照，所以他說他有個朋友想知道這個照片是誰拍的，
因此我跟她朋友就熟了，我記得他是宜蘭高商設計科的，那時候的我對宜商有種莫名的羨慕，
可能是其他的關係吧，反正聊著聊著就熟了，但快接近比賽的時候，我非常的忙，忙到訊息傳過來
必須要好幾個小時之後才能回，而且又狂傳訊息，因此我真的受不了，不再聯繫。

到了兩年後，突然有則訊息傳過來，內容是打：

"其實我很抱歉
那時候一直騷擾你
但拿個時候是因為喜歡你"

這段話深深的影響我很久，讓我意識到太忙或太閒真的都不好。

## 第50屆全國技能競賽 北區賽 ( 4/23 )
---
那時候我們抱著非常緊張的心情去比賽，因為是第一次擔心連分區賽都進不了，那個時候主任跟我說
如果有進去全國賽的話，明年的得獎機會就會比較大一些。

選手在學校是一群非常特別的族群，那時候我是這樣覺得，
出去比賽的前一天，我們搭著遊覽車出去，從校門口出發，
非常的興奮，我們還在遊覽車上打起了傳說對決，
到桃園的時候還在看著窗外剛下課的妹妹。

到了飯店當然開始玩，還記的那次慘痛的經歷，原本7點多要訂肯德基結果到了9點還沒來，
我們真的受不了，直接跟庭瑋下去車站對面買，還有我洗澡的時候奕樊拔我連噴頭 =.=''

* 比賽的時候

那時候是由學長帶我們過去的，搭火車、計程車，直達場地，我們在那邊吃了早餐後，
就直接進入了會場，看到了很多人現在不認識的人，異常的緊張在淘汰賽的時候，
所幸最後是能比分區的，到了隔天真正比賽的時候，我發現我能看到其他人的比賽模式，
於是我就開始施工，做到一半裁判就跟我說做錯了要重來，當下非常的緊張，因為已經落後一大截了
我又把目光轉向學長，真快，那時候只有這樣的想法，看見我旁邊的新北學姊，她好像搞錯了題目要怎麼做
把纜線做成去年的樣子。

到評分環節才是最緊張的時候，聽著其他人對自己作品的預估，然後很漫長的等待，
那時候心裡想，我比完這次就放棄八，到我評分的時候印象很深刻，有個工作人員說，很緊張對八
大家都是一樣的，深呼吸別緊張，這只會讓你亂了腳步。

最後的我分區第三，說實在的，這個成績對那時候的我已經算是非常好的，
因為我覺得自己的練習並沒有很認真，有時候還會睡覺，竟然可以拿第三名。

## 去東南之前(5月~7月)
---

主任突然跟我們說，你們要去東南集訓，心理非常的訝異，
我要離開這裡去台北?
非常的期待八，雖然要離開選手是我非常的難過，但可以去到新環境，
那時候學長安慰我們說，這時候你們要混，因為去了東南大學，
就很難玩起來了，確實我們玩起來了，但就有點放不開。

## 東南之旅(7月~9月)
---

* 全國賽的說明會<br>
這次的說明會也在台中，只不過好像在不同的地方，那時候我們就從宿舍趕往台中，
結束之後再回來宿舍，有夠累的，說明會當中就聽著裁判長跟國手說這次的題目出向，
並且說這次的全國賽會擴大辦理。

去東南大學集訓的時候，是能體會非常特別的感受。

> 在那邊每天早上起來9點去選手室練習，下午5點結束練習，
非常的刻苦，我覺得最困難的是等著時間的流逝八，因為在每天的練習當中，
早就失去了對於布建的熱情，所以之後根本不可能繼續每天的練習。
> 並且還要接受學長的碎念，因此那段時間喪失了對於布建的熱情。

細節來說，去東南的第一周，是非常的想家鄉的，那時候我懂了國中國文老師說的
鄉愁的感覺，懷念家鄉的氣氛、熱鬧程度，在那裏只有冰冷的床板，還有樹林裡的蟲鳴，
回到家有電腦可以玩，還有同學可以聊天，總而言之那兩個月都非常的想，
在練習的時候都覺得過的特別的漫長，希望能早一點回家，有好幾次5點才能離開，
但我們都在2點就全部離開了。

過一個月後我真的受不了，直接把主機電腦搬過去玩，痛苦的不只上午的練習，還有晚上的住宿，
東南的晚上真的有夠破，附近沒有什麼好玩的，又靠近樹林，所以早上結束練習後，晚上又是一個磨練。
話說我最受不了的是把衣服內褲都掛在宿舍裡面，超級不衛生，有一次還撞到他們的內褲......

到最後要離開的時候，我內心是非常興奮的，因為終於要離開這個破地方，
不用在期待會到宜蘭了，我可能未來會懷念著個地方，但一定不是現在。

## 全國賽(10月)
---

這時候其實覺得自己會比的很好，莫名的一種信心，直到進到了比賽會場，發現自己的這份信心
根本就是多餘的，看到別人做那麼多東西並且做的非常快，直接深深的打擊，那三天雖然壓力很大，
但在這麼多人的環境裡面，總有找到一些溫存的感覺，但大家都異常的不說話，
跟會場吵鬧的感覺形成鮮明的對比。

## 全國賽之後 ( 10月~4月 )
---

>在這段時間發生了滿多事，因為學長要比國手賽，所以基本上我們都在空轉，
但我本身很好奇一些機械原理，所以就跟機器人拿了機械手臂來玩，
然後也有跟仁傑學了一些程式語言的概念，總之也是學了非常多的東西，
快到分區比賽的時候我瘋狂的練習，深怕自己又重導去年全國的錯誤，
怕自己會有個時間沒利用到，只要耍費都會讓我身心不安的感覺

細說的話是這樣的：<br>
我們比完全國賽之後，我們想說距離下一次比賽還有很久，並且也要來訓練學弟，
可是那時候峻睿退出了，因為我們都受不了賴公不理性的指揮，認為我們才是最了解目前要做什麼的。
* 學弟<br>
二年級的我們要開始帶學弟了，由於之前都沒有經驗，只能用之前學長帶我們的方式，
可我真的懶得帶學弟，所以都丟給峻睿，所以當他們加入選手，而峻睿退出後，我完全不知道要怎麼和他們溝通，
想到那個時候，學弟都是坐在那邊划手機，而我就練習我的，雖然看了很不爽但也沒什麼辦法能解決這個問題，
學弟的培訓跟我們那時候不一樣，我們是有先學一點網路知識，才接觸過布建，可是這一屆連網路知識都沒有碰，
就開始練習布建。一開始他們是練習製作網路頭，再來是製作網丙的技術崗位，所以雖然在實作上面不成問題，
但是在翻閱題目的時候都會問線路具體是要怎麼布線的，到現在還是會有這個問題。

* 去高雄培訓

那段時間是連休息時間也在想布建要如何施作，
光纖要怎麼佈線，練到成魔了八。

## 分區賽 ( 4月~9月 )
---
在飯店的時候，感覺跟去年的不一樣了，比較的放鬆，還帶著電腦去打osu，
跟學長一起睡在頂樓最好的房間，那次分區是大家最後一次據再一起了，
有些人分區沒上，全國又採異地辦理。
* 比賽前一天的飯店<br>
分區賽前一天，我們選手去到了飯店附近的夜市吃東西，之前都沒有吃過，
回來之後，還聽說學弟們吃到了有老鼠的餐廳，當下我們都覺得很好笑，
希望他們隔天不要拉肚子，走去那個夜市真的很累，回來之後就開始玩電腦，
什麼明天要考淘汰賽的題目都沒有看，雖然心裡還是會有點不安心感，
但比那更強烈的慾望是玩樂跟休息，第二天是我先起床，當我看到學長還在睡的時候，
就先收拾東西了，在飯店大廳等學弟的路上，碰到了電子科主任，他問我說在這幹嘛，
跟他說在等學弟，齁，還要讓學長等學弟，這麼大膽，我也只是笑笑。

* 比賽當天<br>
這次是我憑著之前的映象帶他們去比賽場地，到了那邊發現之前新北的學姊也有來，
他是在去年分區賽和全國賽都有來的，因為看到他很訝異，代表我們隔三代他還在比。
這次參加分區賽並沒有遇到極為困難的地方，只是覺得這次的大家都很友善，
而且都認識，在比賽的時候跟去年全國賽幾乎一樣的施工方式，
所以基本的注意事項跟手感都還留著，印象最深刻就是在我後面比賽的，
時間到一半基本就放棄了，便當直接開吃，而比賽前聽指導老師說，
宜商的選手很厲害，不能輕視。比完賽之後，組長來接我們上遊覽車，
在那時候就覺得自己會得名，而且極有可能是第一名，在休息區的時候大家都有聊一下自己的分數，
而回去的路上我們要先去接機器人還有資網他們，因為他們跟我們是不同的比賽場地，
到場地後看到機器人的心情都滿沮喪的，可能是比賽沒比好，
資網那邊好像也是，之後回去的路上就很開心的睡著了。

* 比賽後的過程<br>
當選手是完全不敢賭定一定會用這條路升學，因為如果成績全部放掉的話，
未來就少一個升學的機會，因此在邊練習選手的時候，還是會讀一下書，
在比完賽後的兩月內，都在補學校的功課，並沒有選手實質上的進展，
主要的進展要到暑假剛開始的時候，主任要求我們把場地搬過去對面的實習處，
因為選手室沒有空間能放置相關的物品，之後就開始暑假的練習時間。
<br>
暑假每天都在練習當中，之前的疫情放了一個月的長假，這讓我擔心自己是不是能在全國賽得名，
放完疫情加後的剛開始練習，我非常反抗練習，因為之前都在家裡睡覺，現在要每天到學校練習，
非常的累，而且冷氣又不冷，在這邊有時候會划手機，有時候就看一下網路上的資訊，
今年的題目都還不知道，所以只能自己幻想題目來做。

* 暑假快結束<br>
練習到現在，發現自己能練習的東西其實並不多，我自己認為銅纜的題目非常重要，
光纖應該大家都一樣爛，於是專研銅纜的題目，光纖的技術並沒有太多，
快接近全國賽的時候，我自己也是跟暑假的練習一模一樣，每天做銅纜，
之後再把它拆掉，每天過這這樣的生活，在那段時間我逐漸意識到，
要與孤單相處，雖然這聽起來是甚麼人生哲學的意義，
但對我來說每天只有自己一個人練習，剛到學校的會去選手室混一下，
有時候到10點才會去練習，晚上準時4點收工，有時候還會提早收工，
檢視我選手生崖真的練習時間都非常的短。晚上會有建築科同屆的人一起練習，
但到了晚上我都會跑過去選手室那邊玩，雖然我有問過指導老師能不能晚上五點就回家，
他也認可了，但還是想在選手室待一會，想最後一次體會選手的生活，
現在的我也覺得非常想念，有空再寫好了。
看著資網出去比賽，我意識到了留給我時間不多了，如果資網得獎只有我沒有，
是不是會很丟臉，當我聽到資網沒有得獎，雖然壓力沒那麼多了，但是更多的是擔心，
如果連這樣都沒有得獎，那我還可以嗎。



在全國賽的集訓當中，搬去了實習處那邊的教室，
感覺跟東南那次很像，都是離開選手室，但至少還能在休息的時候跑過去，
但這次真的是一個人的練習，每天都在那一間教室裡面獨自練習，
還要解決當中的問題，難免有些孤單，因此我就想到可以聽podcase，
確實度過了一段漫長的時間


# 全國賽 ( 10月 )


在比全國賽的時候，我非常的緊張這是最後一次比賽，因為沒有報工科賽，還想說如果沒有得名的話，
那我就回去讀書好了，說不定還能考上好的學校，那時候抱持這樣的心態比賽，

這次比賽的氣氛跟去年完全不一樣，熱熱鬧鬧的雖然只有幾個人，
卻跟去年的完全不依樣，當我們剛開始比賽或是結束的時候都會再一起聊天，
不管是坐在隔壁還是在休息區的時候。
還記的最後一天的時候肚子還非常的痛，去廁所半個小時，還跟宜商的借衛生紙。

* 全國賽
又在一次來到高雄，話說我當選手之後，很常往外跑，
國中完全不常有這種機會，只有待在家的分，現在代表學校出來比賽，
還有想到上一次來到高雄是和海清的一起集訓的樣子，還是一樣的熟悉，
這次的旅館是住更好的，自己一個人的房間，很空曠的，
明天要先去聽一下比賽說明，早一點去好了，
於是我提著探特的心情睡著了。
早上六點多起床，搭著計程車去到了會場，但會場人員跟我說他需要報到通知書，
但我好像沒有帶來，這是我出包的第一次，最後好像用身分證解決，
最後聽裁判助理說：要穿選手服才能進場聽說明會，
這時候我坐不住了，太緊張了，我問新北的學姐說該怎麼辦，
他說那也只好回去飯店拿了，來回搭了1個小時，非常的緊張，
還怕自己趕不上時間，最後到了比賽會場裡面，原來今年場地變這麼大啊，
放完行李之後就開始聽裁判講說比賽的規則以及為甚麼會辦這樣的比賽。
但我覺得那天都只是去交朋友的，在休息時間都和附近的同學聊天，剛好旁邊坐著的是宜商的同學，
就開始問他一些學校的問題，他也認識我們班的同學，以此為話題。


## 第一天
剛開始的第一天連睡覺都睡不好，連晚上睡覺的時候都很難睡，
第一天也是早上6點多起床，清點好物品我就出發了，到了海清發現只有我一個人，
還有它們早上起來的運動隊，邊吃早餐邊看他們晨練，終於選手陸陸續續的來了，
但我意外的發現一件事，我沒又帶到我的工作鞋，這時候又非常的緊張，
還沒比賽就要結束了嗎，我趕快叫著計程車搭回去飯店換鞋子，又在過來，
幸好有即時發現，這是出包的第二次，到了比賽場地感覺跟去年一樣，
先講解題目，然後再比賽。
<br>

中途比到一半，看到其他人也在比，那種感覺就非常的特別，一邊做，也一邊看別人是怎麼做的。
最後結束後，我們也搭著各自的車回家了。

## 第二天
這天就是一個早上的節奏，又是一個把選手牌放在場地裡的節奏，
雖然還是很緊張，怕裁判會不讓我進去，但所幸每個裁判人都很好，
第二天是比光纖，昨天做的工作並沒有落差太大，因此我決定要在今天追回來，
話說今年有兩個海青的選手，看起來不像他們學姊一樣難相處，
都是話非常多的人，他們還會主動找我們攀談。
<br>
做光纖的時候，發生了一件讓我們那個時候的選手群非常訝異的一件事，
就是我們的爆哥(海青的一位學弟)他中途離場，就沒有再回來了，
事後我們打聽到，因為他在規劃光纖的部分犯規了，直接被請出場地。
比賽結束的那個時候，已經是晚上6點了，於是就找到海清的學弟聊天。
聽到賴主任說，他聽說我跟第一名的差距並沒有很大。
